# Catálogo Full Stack — .NET + Nuxt 3

> Desafio técnico **ZDZ Code** — Módulo de Gestão de Catálogo (Produtos e Categorias)

Aplicação full stack para gerenciamento de catálogo comercial com CRUD reativo, integridade referencial e arquitetura containerizada.

## Stack

- **Backend:** .NET 9 · ASP.NET Core Web API · EF Core · xUnit
- **Frontend:** Nuxt 3 · Vue 3 · Vuetify 3 · TypeScript
- **Banco:** SQL Server 2022 (Docker)
- **Orquestração:** Docker Compose
